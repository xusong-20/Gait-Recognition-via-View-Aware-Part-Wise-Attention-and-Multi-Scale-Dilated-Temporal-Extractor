# Gait recognition via View-aware Part-wise Attention and Multi-scale Dilated Temporal Extractor

#### Instructions

1.  Download [code](https://gitee.com/song-xu123/VPA) and dataset

2.   Environment

- Python 3.6
- PyTorch 1.3.1

3. Run

Train a model by
```bash
python train.py
```

Evaluate the trained model by
```bash
python test.py
```
